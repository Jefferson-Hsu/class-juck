## 1.项目介绍

此项目一部分代码基于以前做过的鼠标脚本（定位鼠标坐标），加上后来根据网课规则所写的执行脚本。

## 2.如何使用？

由于该项目基于python开发所以你仅仅只需要安装一个库加上代码的运行即可。

### 安装必要库

 ```shell
pip3 install pyautogui
 ```

### 启动项目

```shell
cd /你下载项目所在的目录
python3 zhenghuo.py
```

## 3.补充内容

## 首先你可以看见项目文件里面拥有一个鼠标定位的工具，你需要运行他然后获取鼠标的位置坐标，然后将鼠标坐标填写到zhenghuo.py这个python文件的18行，然后运行即可！
